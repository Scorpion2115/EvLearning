# Task
Schedule a Pod as follows:
* Name: kucc8
* App Containers: 2
* Container Name/Images:
- nginx
- consul

## Solution
```yml
apiVersion: v1
kind: Pod
metadata:
  name: kucc8
spec:
  containers:
  - name: nginx
    image: nginx
    ports:
    - containerPort: 80
  containers:
  - name: consul
    image: consul
```
